¿QUE ES EL HTML?

HTML fue creado a fines de la década de 1980 y principios de 1990. A través del uso de tags (etiquetas), podemos identificar qué es título dentro del documento, qué es un link que va a otro sitio, qué formato queremos aplicarle como negrita, cursiva, itálica, etc... Como era realmente sencillo, se convirtió en un estándar de internet. Cualquier persona sin tener conocimientos técnicos podían adoptarlo. se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

para comenzar a utilizar html, debemos de instalar el Visual Studio Code, como editor de HTML.

INTRODUCCION AL HTML:TAGS

los TAGS son fragmentos de codigo que permiten crear diferentes elementos en un documento html

< body > para el contenido
< head > para información sobre el documento
< div > división dentro del contenido
< a > para enlaces
< strong > para poner el texto en negrita
< br > para saltos de línea
< H1 >...< H6 > para títulos dentro del contenido
< img > para añadir imágenes al documento
< ol > para listas ordenadas, < ul > para listas desordenadas, < li >
para elementos dentro de la lista
< p > para parágrafos
< span > para estilos de una parte del texto< /p >
